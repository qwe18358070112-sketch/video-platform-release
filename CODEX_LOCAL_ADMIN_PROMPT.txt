【绝对不能搞错的动作路径】
本项目的单路动作路径，最终以这条定义为准：

单击选中 -> 双击放大 -> 停留 dwell_seconds 秒 -> 双击返回 -> 停留 post_restore_dwell_seconds 秒 -> 下一路

你不允许把它简化成：
- 直接双击放大 -> 下一路
- 双击返回后立刻下一路
- 不先单击选中就直接双击

【绝对不能搞错的恢复逻辑】
1. F11 = 返回宫格并重走当前未完成动作路径。
2. F11 不允许直接跳下一路。
3. 自动恢复优先恢复当前路径，只有连续失败达到阈值时才进入冷却跳过。
4. F9 在暂停状态下必须可以连续使用；每按一次都自动选中下一路，但程序保持暂停。
