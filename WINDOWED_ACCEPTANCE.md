# 验收清单（窗口化 / 全屏通用）

## 一、基础轮询
- [ ] 能识别目标窗口
- [ ] 能识别当前是 windowed / fullscreen / auto
- [ ] 能正确切分 12 宫格
- [ ] 单击选中后可双击放大
- [ ] 放大后能停留 `dwell_seconds`
- [ ] 双击返回后能恢复到宫格
- [ ] 返回后能停留 `post_restore_dwell_seconds`
- [ ] 停留结束后才进入下一路

## 二、热键
- [ ] F8 启动
- [ ] F8 暂停
- [ ] F8 继续
- [ ] F9 暂停时连续步进 3 次
- [ ] F10 安全停止
- [ ] F11 恢复到宫格并重走当前路径，不直接跳下一路

## 三、异常恢复
- [ ] 黑屏时记录日志
- [ ] 画面异常时记录日志
- [ ] 自动恢复后优先重走当前未完成路径
- [ ] 连续失败达到阈值后，进入冷却并跳过该路

## 四、布局与顺序
- [ ] 4 宫格正常
- [ ] 6 宫格正常
- [ ] 9 宫格正常
- [ ] 12 宫格正常
- [ ] 12 宫格模板切换正常（例如 twelve_3x4）
- [ ] row_major 正常
- [ ] column_major 正常
- [ ] custom_sequence 正常
- [ ] favorites_name 正常

## 五、人工监管
- [ ] 人工移动鼠标后能暂停
- [ ] 暂停后 F9 可人工巡检下一路
- [ ] F11 后回到当前路径继续巡检

## 六、跨电脑
- [ ] 新电脑能安装依赖
- [ ] 新电脑能完成标定
- [ ] 新电脑能跑 self_test
- [ ] 新电脑能生成发布包
